# BursRadar

BursRadar is an Expo / React Native app for tracking private school scholarship exams, registration periods, and user-specific exam plans.

Main documentation:

- [Project Documentation](docs/project-documentation.md)
- [Google Play Release Checklist](docs/google-play-release.md)

Common commands:

```powershell
npm install
npm run web -- --port 8081
npm run check
npx expo export --platform web
```

Documentation rule: after every meaningful project change, update `docs/project-documentation.md` and its Change Log.
